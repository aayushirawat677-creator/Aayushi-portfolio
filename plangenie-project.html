PlanGenie — AI-Powered Event Planning Platform

An intelligent "genie" for event planning that simplifies discovery, booking, and coordination

Problem Statement

Event planning is fragmented and stressful — users juggle multiple vendors, hidden costs, and coordination headaches. Existing tools serve either weddings (Zola) or corporate events (PlannerNet), leaving Gen Z and SMBs underserved with no comprehensive solution.

Solution

Led the product and marketing strategy for PlanGenie, an AI-powered app that acts as a "genie" for event planning:

Core Platform Features:

  • AI-driven recommendations for vendors and services
  • Integrated vendor marketplace
  • Seamless booking and payment integration
  • Automated coordination and timeline management
  • Real-time collaboration tools

Product Strategy & MVP Definition

MVP Scope Development:

  • Defined comprehensive MVP scope covering discovery → booking → coordination
  • Designed user flows that simplify the entire planning process
  • Created agile roadmap with clear milestones and deliverables
  • Balanced feature richness with development feasibility

User Experience Design:
